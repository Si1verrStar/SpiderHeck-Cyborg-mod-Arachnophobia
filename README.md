This is a mod for SpiderHeck which replaces the spiders with bio-mechanical cyborgs to avoid arachnophobia triggers.



# Build Instructions

First install the Silk mod loader

Then download the mod and open the command prompt, then type: 
dotnet build -c Release

This should create a dll file in bin/Release/net472/
which can be copied into SpiderHeck/Mods/
then launch the game
